# React + Vite
# Node v20.11.1

# Run this project using npm
```
1. npm install
2. npm run dev
```

# Run this project using docker
```
1. go to vite.config.js and change base: "/portfolio-react/" to base: "./portfolio-react/" 
2. cd docker
3. npm run build
4. docker compose up
```

# Deploy this project
```
1. npm run deploy
```
